# online_store
## Rонсольное приложение на Python- Магазин товаров, для управления ассортиментом магазина. 
Основные требования: товары хранятся в списке словарей, каждый товар это словарь с полями(название, цена и кол-во).
Функционал магазина: просмотр всех товаров, добавление товаров, поиск товара по названию, и т.д.
В качестве интерфейса текстовое меню в консоли с возможностью выбора действия.
Этап выполнения: магазин хранится в виде списка продуктов, где каждый элемент-словарь, у словаря поля (название, цена и кол-во)в качестве ключей, и реализуется цикл wile(true), выводиться меню:
1. Показать все товары
2. Добавить товар
3. Найти товар
4. Удалить товар
5. Общая стоимость
0. Выход

Используется оператор input, вводится число в соответствии с меню(0- выход; 2- добавить товар; 5-общая стоимость и т.д).
также используются операторы if, elif, else для обработки этого выбора.  
И каждое действие - 'Найти товар', 'Общая стоимость' и т.д оформлено в отдельную функцию(класса). В работа со строками и списками, при поиске и удалении используется метод строк: Lower.

ООП. Создаётся класс Product с атрибутами: name, price, quantity и методами для изменения этих данных.
Класс Store, где список этих товаров.
